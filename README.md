# DrishyaMitra

AI-powered photo management system with face recognition and intelligent search.

## Quick Start

### Option 1: Docker (Recommended - Easiest!)

1. Install Docker Desktop
2. Run: `docker-start.bat`
3. Open: http://localhost:3000

See [DOCKER_SETUP.md](DOCKER_SETUP.md) for details.

### Option 2: Manual Setup

See [SETUP_GUIDE.md](SETUP_GUIDE.md) for manual installation.

## Features

- 📸 Photo upload and management
- 👤 Automatic face detection and recognition
- 🔍 AI-powered photo search
- 💬 Chat with your photos using natural language
- 📧 Email delivery of photos
- 🏷️ Face labeling and people management

## Tech Stack

- Backend: Python, Flask, PostgreSQL, DeepFace, TensorFlow
- Frontend: React, Vite, TailwindCSS
- AI: Groq API for chat, DeepFace for face recognition
